## Terminal & Shell
- [zsh](https://www.zsh.org/)
- [Alacritty](https://alacritty.org/) - Terminal emulator
- [tmux](https://github.com/tmux/tmux) - Terminal multiplexer
- [sheldon](https://github.com/rossmacarthur/sheldon) - Zsh plugin manager
- [fzf-tab](https://github.com/Aloxaf/fzf-tab) - Zsh completion with fzf

## CLI Tools
- [Neovim](https://neovim.io/)
- [ripgrep](https://github.com/BurntSushi/ripgrep) - Fast grep replacement
- [fd](https://github.com/sharkdp/fd) - Fast find replacement
- [bat](https://github.com/sharkdp/bat) - cat/less with syntax highlighting
- [eza](https://github.com/eza-community/eza) - Modern ls replacement
- [dust](https://github.com/bootandy/dust) - Intuitive du replacement
- [zoxide](https://github.com/ajeetdsouza/zoxide) - Smarter cd
- [bottom](https://github.com/ClementTsang/bottom) - System monitor
- [newsboat](https://newsboat.org/) - RSS reader
- [nyaa](https://github.com/Beastwick18/nyaa) - TUI for nyaa.si
