Core Concepts

Data-Oriented Design

Understanding how data flows through your system and optimizing for cache locality and memory access patterns.

  • CppCon 2014: Mike Acton "Data-Oriented Design and C++" - Foundational talk on DoD principles
  • Andrew Kelley Practical Data Oriented Design (DoD) - Practical applications beyond C++
  • The Lost Art of Structure Packing - Memory layout optimization
  • Handles vs Pointers - Data structure design patterns
  • Memory access is O(N^[1/3]) - Understanding real-world memory performance
